# 青翼读书会

青翼读书会是一个以阅读活动为核心的社区小程序。用户可以通过签到积攒积分、浏览和报名线下读书活动、获取好书推荐；管理员可以通过 Web 后台管理用户、活动、订单和内容。

---

## 快速开始

### 环境准备

- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)（稳定版）
- Node.js 18+

### 1. 导入小程序项目

1. 打开微信开发者工具，选择「导入项目」
2. AppID 填写
4. 点击确定，等待编译完成即可预览

### 2. 开通云开发

1. 在微信开发者工具中点击「云开发」按钮
2. 开通云开发环境，记录环境 ID
3. 打开 `miniprogram/app.js`，将 `cloud.init` 中的环境 ID 替换为你的环境 ID：
   ```js
   cloud.init({ env: '你的环境ID' })
   ```

### 3. 部署云函数

1. 在微信开发者工具左侧文件树中展开 `cloudfunctions/` 目录
2. 右键点击每个云函数目录（`user`、`event`、`pay`、`admin`）
3. 选择「上传并部署：云端安装依赖」
4. 等待部署完成（首次约 1-2 分钟）
5. **注意**：`event` 云函数的 `config.json` 声明了生成小程序码的权限，部署后需等待约 10 分钟生效

### 4. 导入测试数据

项目提供了 Mock 数据用于开发调试，位于 `.vibe-context/mock_data.js`。

导入方式：
1. 打开云开发控制台 -> 数据库
2. 按需创建以下集合：`users`、`events`、`registrations`、`orders`、`point_logs`、`comments`、`banners`、`settings`、`books`、`notifications`
3. 向各集合中导入 `.vibe-context/mock_json/` 目录下对应的 JSON 文件

详细的数据库字段定义见 `.vibe-context/data_schema.md`。

## 小程序使用方法

小程序底部有四个 Tab：首页、活动、商城（规划中）、我的。

### 首页

- **签到**：点击签到按钮获得积分奖励，连续签到有额外加成
- **热门活动**：展示近期可报名的活动卡片，点击进入活动详情
- **好书推荐**：精选图书推荐，点击查看书籍详情页

### 活动大厅

活动大厅分为两个 Tab：

- **活动列表**：浏览所有可报名的活动
- **我的活动**：查看已报名的活动，展示核销码（二维码入场凭证）

#### 活动报名模式

活动支持三种报名方式，由活动创建时设定：

| 模式 | 说明 |
|------|------|
| 免费报名 | 直接报名，无需支付 |
| 积分兑换 | 消耗指定积分兑换名额 |
| 付费报名 | 通过微信支付购买名额，支持 15 分钟内取消退款 |

报名成功后会生成专属核销码，活动现场出示给管理员扫码即可入场。

### 个人中心

- **资料卡片**：头像、昵称、会员等级
- **积分余额**：当前可用积分，点击查看积分明细
- **等级进度**：显示距离下一等级的进度
- **我的活动**：已报名活动列表及核销码
- **我的订单**：付费活动的支付记录和退款状态
- **通知中心**：系统消息和活动提醒
- **资料编辑**：修改头像、昵称、手机号

### 积分体系

- **签到奖励**：每日签到获得基础积分，连续签到有额外奖励
- **活动奖励**：参加活动后获得奖励积分
- **等级晋升**：累计积分达到阈值自动升级（青铜 -> 白银 -> 黄金）
- **积分消费**：用于积分模式报名活动

### 管理员功能

管理员在小程序中可以：
- **扫码核销**：在「我的」页面进入核销功能，扫描参与者的核销码完成签到

## Web 管理后台

### 启动开发

```bash
cd web-admin
npm install
npm run dev
```

浏览器访问本地开发地址即可。生产环境需 `npm run build` 后将 `dist/` 目录上传到云开发静态网站托管。

### 登录

管理员使用云开发邮箱密码登录。管理员账号需在云开发控制台预先创建。

### 功能模块

#### 数据概览（Dashboard）

- 用户总数、活动数量、订单金额等核心指标
- 近期趋势图表

#### 用户管理

- 用户列表：查看所有注册用户
- 积分调整：手动增减用户积分（附备注原因）
- 用户详情：查看积分流水、报名记录

#### 活动管理

- 创建/编辑活动：设置标题、时间、地点、报名模式（免费/积分/付费）、人数上限
- 报名管理：查看报名列表、核销状态
- 活动状态控制：发布、暂停、结束

#### 订单管理

- 订单列表：查看所有付费订单及状态
- 退款审批：用户申请退款后在此审批通过或拒绝
- 订单详情：支付信息、退款记录

#### 内容管理

- 首页轮播图：上传/排序/启用禁用
- 好书推荐：添加/编辑推荐图书

#### 系统设置

- 积分规则：签到奖励、连续签到加成
- 等级配置：各级别名称和积分阈值
- 订单超时：未支付订单自动取消时间
