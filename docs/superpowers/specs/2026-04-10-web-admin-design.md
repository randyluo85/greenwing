# Web 管理后台设计文档

**项目**: 青翼读书会 - Web 管理后台
**日期**: 2026-04-10
**状态**: 待实现

---

## Context

青翼读书会小程序的前端原型已完成（prototype/ 文件夹），PRD 和数据表定义已就绪（.vibe-context/）。现在需要构建 Web 管理后台，让管理员在电脑端完成用户管理、活动管理、订单管理、内容管理和系统配置。

本阶段只构建前端页面骨架和交互，不对接云函数/数据库。数据使用 mock。

---

## 技术方案

- **框架**: Vue 3 + Vite
- **UI 库**: Element Plus
- **路由**: Vue Router 4
- **HTTP/数据**: 暂用本地 mock 数据，后续切换到 @cloudbase/js-sdk
- **富文本**: wangeditor 或 quill（活动详情编辑）
- **导出**: xlsx（报名名单/订单导出）
- **样式**: 延续小程序原型 Emerald Modern 配色（主色 #00897B）

---

## 整体布局

经典左侧导航 + 顶部工具栏：

- **左侧导航栏** (190px): 深色背景 (#00695C)，Logo + 6 个导航项
- **顶部工具栏** (48px): 当前页面标题 + 日期 + 管理员头像
- **内容区**: 浅灰背景 (#f0f2f5)，白色卡片承载内容

导航菜单:
1. Dashboard
2. 用户管理
3. 活动管理

## 6 大模块详细设计

### 1. Dashboard (Dashboard.vue)

**定位**: 社区活跃度概览，不是营收看板。

**统计卡片** (4 列):
| 卡片 | 数据 | 辅助信息 |
|------|------|----------|
| 社区成员 | 总用户数 | 本周新增 |
| 本周签到 | 签到人次 | 日均 |
| 本月活动参与 | 报名人次 | - |
| 累计收入 | 金额 | 右上角「月/年/全部」切换 |

**图表区** (两列):
- 左: 社区活跃趋势（柱状图，签到+参与双维度，周/月切换）
- 右: 近期活动报名进度（进度条列表）

**底部** (左3右2):
- 左: 最近动态时间线（报名/签到/核销/退款等事件流）
- 右: 会员等级分布（青铜/白银/黄金占比 + 积分发放/消耗统计）

### 2. 用户管理 (Users.vue)

**筛选栏**: 会员号/昵称/手机号搜索 + 等级下拉 + 角色下拉 + 注册时间范围

**用户表格**:
| 列 | 说明 |
|----|------|
| 会员号 | QYxxx 格式，绿色高亮 |
| 用户 | 头像 + 昵称 |
| 手机号 | 脱敏显示 |
| 等级 | 颜色标签（青铜橙/白银灰/黄金金） |
| 积分 | 当前积分 |
| 角色 | 标签（管理员绿/核销员红/普通灰） |
| 注册时间 | yyyy-MM-dd |
| 操作 | 调整积分 / 编辑 |

**交互 - 调整积分弹窗**:
- 显示当前积分
- 增加/扣减 单选切换
- 积分数额输入
- 调整原因（必填）
- 确认/取消

**交互 - 编辑用户弹窗**:
- 昵称输入
- 角色下拉（普通用户/核销员/管理员）
- 等级下拉（青铜/白银/黄金）
- 操作提示警告

### 3. 活动管理 (Events.vue)

**状态 Tab**: 全部 / 已发布 / 草稿 / 已结束（带数量角标）

**活动列表**: 卡片式布局，每张卡片包含:
- 左: 尖叫图缩略图
- 右: 标题 + 状态标签 + 报名模式标签（免费/积分/付费）+ 时间地点主讲 + 报名人数进度 + 核销数/收入
- 操作: 报名名单 / 编辑（草稿多一个删除）

**交互 - 创建/编辑活动（右侧抽屉 520px）**:
分三段:
1. **基本信息**: 标题、分类、人数、活动时间、报名截止、地点、主讲人、封面图上传、富文本编辑器
2. **报名模式**: 三选一（免费/积分/付费），选择积分时显示积分数额和等级门槛，选择付费时显示价格和退款政策
3. **其他设置**: 活动奖励积分、初始状态（草稿/发布）
- 底部: 取消 / 保存草稿 / 发布活动

**交互 - 报名名单（右侧抽屉 520px）**:
- 顶部统计: 已报名 / 已核销 / 已取消 / 剩余名额
- 表格: 用户头像+昵称 / 报名方式 / 报名时间 / 核销状态
- 右上角: 导出 Excel 按钮

### 4. 订单管理 (Orders.vue)

**状态 Tab**: 全部 / 已支付 / 待退款（红色数量角标）/ 已退款 / 已关闭

**筛选栏**: 订单号/用户名搜索 + 活动下拉 + 日期范围 + 导出按钮

**订单表格**:
| 列 | 说明 |
|----|------|
| 订单号 | ORDxxx 格式，绿色 |
| 用户 | 昵称 |
| 活动 | 活动名 |
| 金额 | 元 |
| 状态 | 彩色标签 |
| 支付时间 | MM-dd HH:mm |
| 操作 | 审批(待退款) / 详情 |

待退款行黄色背景高亮。

**交互 - 退款审批弹窗**:
- 订单信息网格（订单号/用户/活动/金额/支付时间/支付方式）
- 用户退款原因（黄色高亮卡片）
- 退款政策提示（绿色提示条）
- 审批备注文本框
- 按钮: 取消 / 驳回（红色边框）/ 同意退款（绿色）

**交互 - 订单详情弹窗**:
- 状态标签 + 订单号
- 信息网格（用户/活动/金额/微信交易号/支付时间/过期时间）
- 操作记录时间线

### 5. 内容管理 (Content.vue)

**子 Tab**: 轮播管理 / 推荐图书

**轮播管理**:
- 列表式，每行: 序号 + 缩略图 + 标题 + 跳转链接 + 状态标签(上线/下线) + 上下移动 + 编辑 + 上线/下线
- 已下线项降低透明度
- 右上角: 添加轮播按钮

**推荐图书**:
- 三列卡片网格，每张卡片: 封面缩略图 + 书名 + 作者 + 简介(两行截断) + 编辑/删除
- 右上角: 添加图书按钮

### 6. 系统设置 (Settings.vue)

单页表单，无 Tab，分三个卡片:

**签到积分规则**:
- 每日签到基础积分（默认 10）
- 连续签到额外奖励（默认 5）
- 右上角: 重置默认

**会员等级配置**:
- 三行，每行: 等级色块 + 等级名 + 积分范围（下限显示文本 + 上限可编辑输入框）
- 黄金会员上限为"不限"

**订单设置**:
- 订单支付超时时间（默认 15 分钟）

底部: 保存设置按钮
