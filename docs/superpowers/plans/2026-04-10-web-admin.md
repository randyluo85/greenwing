# Web 管理后台实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建青翼读书会 Web 管理后台的完整前端骨架（6 个模块页面 + 交互弹窗/抽屉），使用 mock 数据，不对接后端。

**Architecture:** Vue 3 + Vite 单页应用，Element Plus 组件库，Vue Router 管理路由。经典左侧导航布局，所有业务数据来自 `src/mock/` 目录的静态 JSON。组件按职责拆分：页面级 views + 可复用弹窗/抽屉 components。

**Tech Stack:** Vue 3 (Composition API, `<script setup>`) + Vite + Element Plus + Vue Router 4

```js
/**
 * 格式化日期为 yyyy-MM-dd
 */
export function formatDate(date) {
  if (!date) return ''
  const d = new Date(date)
  const y = d.getFullYear()
  const m = String(d.getMonth() + 1).padStart(2, '0')
  const day = String(d.getDate()).padStart(2, '0')
  return `${y}-${m}-${day}`
}

/**
 * 格式化日期为 MM-dd HH:mm
 */
export function formatDateTime(date) {
  if (!date) return ''
  const d = new Date(date)
  const m = String(d.getMonth() + 1).padStart(2, '0')
  const day = String(d.getDate()).padStart(2, '0')
  const h = String(d.getHours()).padStart(2, '0')
  const min = String(d.getMinutes()).padStart(2, '0')
  return `${m}-${day} ${h}:${min}`
}

/**
 * 金额分转元
 */
export function fenToYuan(fen) {
  if (fen == null) return '0.00'
  return (fen / 100).toFixed(2)
}

/**
 * 手机号脱敏 138****5678
 */
export function maskPhone(phone) {
  if (!phone || phone.length < 7) return phone
  return phone.slice(0, 3) + '****' + phone.slice(-4)
}

/**
 * 格式化金额显示 ¥xx.xx
 */
export function formatMoney(fen) {
  return '¥' + fenToYuan(fen)
}
```
